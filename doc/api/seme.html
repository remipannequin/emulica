seme is an emulation package oriented towards modelling of industrial 
systems from a logistics point of view. Is it based on the SimPy simulation framework.

Contains the following modules:
 
emulation -- enable to model and emulate a system (based on SimPy)
gui -- usefull classes to implement visual emulation (based on goocanvas and gtk)
plot -- utility to create various plot from simulation results
xml -- reading and writing xml-based model files
